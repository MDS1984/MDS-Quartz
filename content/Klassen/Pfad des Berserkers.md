# Raserei Auslöser
Woher stammt deine wilde Kraft und die Fähigkeit, Schmerz zu ertragen? Wähle zwei Ursprünge, vereine sie zu deiner Quelle – und einen, den du verachtest.
- Rituelle Vorbereitung
- Visionen von schrecklichen oder heldenhaften Szenen
- Antrainierte Schmerzresistenz
- Tiefe meditative Konzentration
- Langer Konsum seltsamer Substanzen
- Blutpakt mit einer verwandten Macht
- Kraft aus den Geistern der Ahnen schöpfen
- Unzählige Schlachten, unzählige Verletzungen
- Urkraft aus der Wildnis ziehen
- Die eigene Seele an ein physisches Totem binden
- Erinnerungen kanalisieren, die du niemals vergessen wirst
- Du weißt wirklich nicht, warum du so bist
# Narben
Deine Fähigkeit, Schmerz zu ignorieren, hat dich weit mehr Schaden erleiden lassen als die meisten Krieger. Welche Narben trägst du, die deine Rücksichtslosigkeit offenbaren? Wähle drei, die du stolz zeigen kannst:

|                               |                                |                                       |
| ----------------------------- | ------------------------------ | ------------------------------------- |
| Gebrochene, schiefe Nase      | Wildes, zerzaustes Haar        | Zerissene Ohrläppchen oder Ohrspitzen |
| Netz aus tiefen Schnittnarben | Muskeln wie aus Eisen geknotet | Tiefe, ausgebrannte Wunde             |
| Dauerhaft gespaltene Knöchel  | Mehrere fehlende Finger        | Gesprungene Kieferlinie               |
| Leere Augenhöhle              | Zackige Brandnarben            | Zertrümmerter Fuß                     |
| Zerschlagene Zähne            | Herausgerissene Krallenspuren  | Eingeschlagener Schädel               |
| Blumenkohlohren               | Runen-Narben durch Magie       | Keine einzige Schramme                |
