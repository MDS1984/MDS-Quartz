# Signatur Bewegungen
Du hast einen einzigartigen und einprägsamen Stil. Was tust du, das andere fesselt? Wähle 3 Dinge, an die sich andere bei dir erinnern – oder denke dir eigene aus.

|                                    |                                          |                                          |
| ---------------------------------- | ---------------------------------------- | ---------------------------------------- |
| Feinde verhöhnen                   | Haare und Outfit mitten im Kampf richten | Versuchen, das Publikum zu beeindrucken  |
| Schummeln und täuschen             | Gegner demütigen                         | Einen Spruch sagen                       |
| Witze und Beleidigungen schleudern | Hut tippen oder Umhang schwenken         | Initialen am Tatort hinterlassen         |
| Identität verbergen                | Sich selbst absichtlich einschränken     | Ein Kostüm tragen                        |
| Lässige Haltung einnehmen          | In der dritten Person von sich sprechen  | Pirouetten drehen und Luftsprünge machen |
| Ein Getränk in der Hand halten     | Münzen, Würfel oder Karten werfen        | Duelle romantisieren                     |
# Vergangene Verwicklungen
Ein Draufgänger ist bekannt für seine Vergangenheit. Wessen Leben ist mit der Geschichte deiner Taten verknüpft? Wähle 2 Personen, die in deinen Erzählungen wichtig sind, indem du je eine von jeder Seite auswählst und sie miteinander kombinierst. Wähle eine weitere Person, die für deine Geschichte von Bedeutung ist, die du aber noch nicht kennst.

| -------------- | ------------ | ------------- | -------------- | ------------ | --------------- |

# Credo
Viele aufregende Draufgänger haben einen Kodex – eine Regel, die sie niemals brechen, eine Grenze, die sie nie überschreiten, oder ein Sprichwort, nach dem sie ihr Leben führen. Wie sehr bist du diesem Kodex verpflichtet? Wähle bis zu zwei unten aus oder denke dir deinen eigenen aus.

|                                         |                                              |
| --------------------------------------- | -------------------------------------------- |
| In Liebe und Krieg ist alles erlaubt    | Ehre ist für die Toten                       |
| Führe das Schwert der Gerechtigkeit     | Eine gute Geschichte ist jede Belohnung wert |
| Nimm jede Herausforderung an            | Töte niemals die Hilflosen                   |
| Münzen sind zum Ausgeben da             | Lieber der Tod als die Unehre                |
| Berüchtigt ist auch berühmt             | Der beste Plan ist kein Plan                 |
| Ein Gentleman selbst in schweren Zeiten | Nur ein fairer Kampf zeigt wahres Können     |
